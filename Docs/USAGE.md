# 使用指南

## 快速开始

### 1. 安装插件

1. 下载最新版本
2. 双击安装包，按提示完成安装
3. 启动 Word，在 COM 加载项中启用 "GOWordAgent"

### 2. 配置 AI 服务

1. 在 Word 右侧边栏点击 **设置** 标签
2. 选择 AI 提供商：
   - **DeepSeek**: 输入 API Key，使用默认地址
   - **智谱 GLM**: 输入 API Key，使用默认地址
   - **Ollama**: 输入本地服务地址（默认 http://localhost:11434）
3. 点击 **保存并连接**
4. 勾选 **自动连接** 可让插件启动时自动连接

### 3. 使用 AI 问答

1. 在底部输入框输入问题
2. 点击发送按钮或按 Enter
3. AI 回复会显示在聊天区域

### 4. 文档校对

#### 精准校验

适合：检查错别字、语病、术语不一致

1. 打开需要校对的文档
2. 在顶部点击 **纠错（审阅）**
3. 等待校对完成
4. 在 Word **审阅** 面板中查看修订建议
5. 逐条接受或拒绝修订

#### 全文校验

适合：系统性检查语法、标点、用词、逻辑

1. 在设置中选择 **全文校验** 模式
2. 可自定义提示词（高级用户）
3. 点击 **纠错（审阅）**

## 高级功能

### 缓存机制

- 校对结果会自动缓存（基于文本内容 SHA256 哈希）
- 相同内容再次校对时从缓存读取，几乎瞬间完成
- 缓存仅在当前 Word 会话有效，关闭 Word 后清空
- 点击 **清空** 按钮可手动清除缓存

### 增量校对

- 修改文档后再次校对，只会处理变更的段落
- 未变更的段落从缓存读取结果
- 大幅提高重复校对效率

### 多文档支持

- 支持同时打开多个 Word 文档
- 插件自动绑定当前活动文档
- 切换文档时会自动重新绑定
- 每个文档的校对结果独立

## 系统要求

### 支持的 Office 版本

| 版本 | 支持状态 | 备注 |
|------|:--------:|------|
| Microsoft 365 | ✅ | 完全支持 |
| Word 2021/2019/2016 | ✅ | 完全支持 |
| Word 2013 | ✅ | 部分功能降级 |
| Word 2010 | ⚠️ | 修订模式降级为普通替换 |
| WPS | ❌ | 不支持 |

### 兼容性说明

**Word 2010 已知限制**:
- `ScrollIntoView` 方法可能不可用（定位时不会自动滚动）
- `TrackRevisions` 失败时自动降级为普通文本替换（无修订标记）
- 批注功能正常可用

**Word 2013 已知限制**:
- `wdMixedRevisions` 模式不可用
- 其他功能正常使用

## 故障排除

### 无法连接 AI 服务

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| API Key 无效 | 密钥错误或过期 | 检查并重新输入 API Key |
| 连接超时 | 网络问题或服务不可用 | 检查网络连接，稍后重试 |
| 请求过于频繁 | 超出 API 速率限制 | 等待几分钟后重试 |

### 校对结果异常

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 某段长时间无响应 | 该段内容复杂，AI 推理慢 | 等待或取消后重试 |
| 修订位置偏移 | 文档在校对过程中被修改 | 重新打开文档，从头校对 |
| 表格内容混乱 | 表格文本提取顺序问题 | 暂时将表格内容转为普通文本校对 |
| 定位失败 | 文档已切换或关闭 | 确保目标文档是当前活动文档 |

### 性能优化

**大文档处理：**
- 文档 > 3000 字会自动分段处理
- 每段约 1500 字，100 字重叠防止边界遗漏
- 默认 5 段并发，可在代码中调整 `concurrency` 参数

**内存占用：**
- 缓存最多保留 1000 段结果
- 每段缓存约 1-5KB
- 超大文档（>50MB）建议分批处理

**高分辨率屏幕：**
- 已添加 DPI 感知配置
- 如仍出现模糊，检查 Windows 显示缩放设置

### 多文档使用

**问题**: 切换文档后校对结果应用到错误文档
**解决**: 
1. 确保目标文档是当前活动窗口（点击文档标题栏）
2. 等待 1-2 秒让插件检测文档切换
3. 重新点击"纠错（审阅）"

### 隐私与安全

**配置加密**:
- API Key 使用 Windows DPAPI 加密存储
- 配置文件包含 HMAC-SHA256 完整性验证
- 配置文件路径: `%AppData%\GOWordAgentAddIn\config.dat`

**隐私同意**:
- 首次使用校对功能时显示隐私提示
- 每天首次使用时重新提示
- 文档内容会发送到 AI 服务商

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Enter | 发送消息 |
| Ctrl + Enter | 换行 |

## 注意事项

1. **保存文档**: 校对前请确保文档已保存
2. **备份重要文档**: 建议在校对前创建备份
3. **API 费用**: 使用在线 AI 服务会产生 API 调用费用
4. **隐私**: 文档内容会发送到 AI 服务商，注意敏感信息
5. **兼容性**: 不支持 WPS Office，仅支持 Microsoft Word
